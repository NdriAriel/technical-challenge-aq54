AQ54 Data Visualization App with Angular and Docker-Compose and Nodejs

Overview

This repository contains the source code and configuration files for a Data Visualization App built with Angular and Docker-Compose. The app is designed to visualize and analyze data in a user-friendly manner.

Prerequisites

Before running the application, make sure you have the following software installed on your machine:

  • Docker
  • Docker-Compose
  • Node.js

Getting Started

  1. Clone the repository:

    Example :
    git clone https://github.com/yourusername/data-visualization-app.git
cd data-visualization-app

  2. Install Angular CLI globally:

    Example :
    npm install -g @angular/cli

  3. Install project dependencies:

    Example :
    npm install

  4. Build the Angular app:

    Example :
    ng build

  5. Start the Docker containers:

    Example :
    docker-compose up -d

  6. Open your web browser and navigate to http://localhost:8080 to access the Data Visualization App.

Project Structure

  • src/: Contains the Angular application source code.
  • docker-compose.yml: Configuration file for Docker-Compose.
  • Dockerfile: Dockerfile for building the Angular app image.

Docker-Compose Configuration

The docker-compose.yml file defines the services, networks, and volumes needed for the application. It includes the following services:

  • web-app: Angular app service.
  • nginx: Nginx web server for serving the Angular app.
  • api: Placeholder for any backend API service.
  • database: Placeholder for any backend database service.

Development Workflow

  1. Make changes to the Angular app source code in the src/ directory.

  2. Rebuild the Angular app:

    Example :
    ng build

  3. Restart the Docker containers:

    Example :
    docker-compose restart web-app

  4. Access the updated app at http://localhost:8080.
